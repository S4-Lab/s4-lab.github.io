# Lab 1: Blinking LED

This lab aims to familairize you with the STM32F4Discovery board and the STM32F411 microcontroller. Learn how to set up the development environment to program and flash the code on the STM32F411.

### Resources
* Discovery kit with STM32F411VE MCU
* Reference Manual
* Programming Manual of STM32 Cortex-M4 MCUs

## Part 1: Setting the development environment
* Install the [GNU ARM Embedded Toolchain](https://developer.arm.com/downloads/-/arm-gnu-toolchain-downloads) for compiling baremetal code.
* (For Windows user) Install **make** tool from [gnuwin32](https://gnuwin32.sourceforge.net/packages/make.htm) (alternatively, [mingw](https://www.mingw-w64.org/) or [msys2](https://www.msys2.org/)).
* Install either [stlink](https://github.com/stlink-org/stlink) or [OpenOCD](https://openocd.org/) for programming and debugging the code.

Easy install: Mac users can use `homebrew` and Linux (Ubuntu) users can use `apt` to easily install the above packages.
